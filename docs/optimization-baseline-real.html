Cross-check vs static estimate

The static chars/3.8 estimate in optimization-baseline.html measures plugin file size, not run cost. Compare:

Static estimate (whole plugin worst-case sum)~12,003 + 12,298 + 4,238 + 786 = ~29,325 tok
Real-run effective input + output (pre-trim)23,388,215 tok

The real-run number is far higher than static because: (a) every turn re-loads context (compounded by cache), (b) intermediate tool results count toward input, (c) prompt/response cycles include user content beyond the plugin. The static estimate bounds plugin contribution per turn; the real number captures total session economics.
